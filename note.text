ENUGUAIR TRAVELS WEBSITE DOCUMENTATION
CSS GROUP PROJECT – STREAM 2
GROUP NUMBER 15

1.0 PROJECT OVERVIEW
The EnuguAir Travels Website is a responsive multi-page travel agency website created to encourage users to explore travel destinations and book vacation experiences. The website was developed using HTML5 and CSS3.
The project focuses on responsive design, modern layouts, consistent styling, and user-friendly navigation across all pages.
